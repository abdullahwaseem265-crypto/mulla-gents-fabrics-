# Wasay Jents Fabrics - Website Specification

## 1. Project Overview
- **Project Name:** Wasay Jents Fabrics
- **Type:** Single-page cloth/fabric store website
- **Core Functionality:** Showcase premium fabrics and clothing materials with elegant design
- **Target Users:** Customers looking for quality fabrics, fashion designers, tailors

## 2. UI/UX Specification

### Layout Structure
- **Header:** Fixed navigation with logo, menu links
- **Hero Section:** Full-width banner with tagline and CTA
- **Collections:** Grid display of fabric categories
- **Featured Products:** Carousel/grid of popular fabrics
- **About Section:** Brand story and values
- **Contact Section:** Contact form and store information
- **Footer:** Social links, quick links, copyright

### Visual Design

#### Color Palette
- **Primary:** #1a1a2e (Deep Navy) - Luxury and elegance
- **Secondary:** #c9a227 (Gold) - Premium feel
- **Accent:** #8b4513 (Saddle Brown) - Fabric/textile feel
- **Background:** #faf8f5 (Off-white) - Clean and sophisticated
- **Text Dark:** #2d2d2d
- **Text Light:** #ffffff

#### Typography
- **Logo/Headings:** "Playfair Display" (Serif - elegant)
- **Body Text:** "Raleway" (Sans-serif - clean)
- **Accent Text:** "Cormorant Garamond" (Refined)

#### Logo Design
- Text-based logo: "WJF" monogram + "Wasay Jents Fabrics"
- Icon: Elegant thread/fabric-inspired monogram
- Colors: Gold on navy or navy on white

### Components
- Navigation bar with smooth scroll
- Hero banner with parallax effect
- Product cards with hover effects
- Contact form with validation
- Social media links
- Mobile-responsive hamburger menu

## 3. Functionality Specification

### Core Features
1. **Responsive Navigation** - Works on all devices
2. **Smooth Scrolling** - Anchor links scroll smoothly
3. **Product Showcase** - Display fabric collections
4. **Contact Form** - Functional form (client-side validation)
5. **Animated Elements** - Subtle animations on scroll
6. **Custom Favicon** - Based on logo design

### User Interactions
- Hover effects on buttons and cards
- Smooth scroll navigation
- Form validation feedback
- Mobile menu toggle

## 4. Acceptance Criteria
- [ ] Website loads without errors
- [ ] Logo/icon displays correctly in browser tab
- [ ] All sections are properly styled
- [ ] Mobile responsive design works
- [ ] Contact form validates inputs
- [ ] Smooth scrolling works for navigation
- [ ] Color scheme matches specification
